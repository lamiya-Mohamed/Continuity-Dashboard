# Continuity Dashboard

**وصف المشروع:**  
هذا المشروع عبارة عن **تطبيق تفاعلي باستخدام Streamlit** لمتابعة **جاهزية المجالات المختلفة** في أي منظمة أو مشروع.  
يمكن للمستخدم:  
- إضافة مجالات جديدة مع اسمها، درجة الجاهزية، والوصف.  
- عرض جميع المجالات الحالية.  
- تعديل درجة الجاهزية لأي مجال.  
- رسم Bar Chart يوضح درجات الجاهزية لكل المجالات.  
- حفظ البيانات في ملف وقراءتها من ملف.  
- حذف الملفات الغير مرغوب فيها.  

**تقنيات المشروع:**  
- Python 3  
- Streamlit  
- Matplotlib
